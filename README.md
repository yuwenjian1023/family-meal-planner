# 家庭食谱计划助手

一个帮助家庭管理食谱、食材库存和饮食计划的 PWA 应用。

## 功能特性

- 🍳 **食谱浏览**：200+ 道精选食谱，支持搜索和分类筛选
- 📅 **饮食计划**：按日期安排三餐，智能推荐
- 🛒 **食材库存**：管理家中食材，过期提醒
- 🛍️ **购物清单**：根据饮食计划自动生成采购清单
- 👨‍👩‍👧‍👦 **家庭共享**：支持多家庭管理，邀请码加入
- ☁️ **云端同步**：数据自动同步到 Supabase

## 技术栈

- React + TypeScript + Vite
- Zustand (状态管理)
- Supabase (后端 + 认证)
- Tailwind CSS (样式)

## 开发

```bash
npm install
npm run dev
```
